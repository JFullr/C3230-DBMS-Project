As an Administrator:

= As an administrator, I would like to be able to log into the database 
so that I can read and write records directly as needed.
= As an administrator, I want to execute SQL queries directly 
so that I can read and write records directly as needed.
= As an administrator, I want to be able to view the results of my SQL queries as a table 
so that I can read the records from the database.
= As an administrator, I want to be able to view a log of any changes made to the database 
so that the security of the system can be maintained and allow for auditing.

As a Nurse:
= As a nurse, I want to be able to log in to the system 
so that I can work with the records in the database.
= As a nurse, I want to be able to create new patient records 
so that a record of the patient's care is available for future reference.
= As a nurse, I want to be able to setup appointments for patients 
so that patients, nurses, and the doctors know when the patient will come to their appointment.
= As a nurse, I want to be able to edit patient information 
so that all patient information is accurate and up-to-date.
= As a nurse, I want to be able to input initial diagnoses 
so that the patient’s record is up-to-date and allows for further follow-up if required.
= As a nurse, I want to be able to search for patients by their information 
so that I can register new appointments or modify their information.
= As a nurse, I want to be able to edit appointments 
so that I can reschedule appointments or make accommodations for the patient as necessary.
= As a nurse, I want to be able to view visit information 
so that I have a prior record of the patient's visits to the client.
= As a nurse, I want to be able to edit any initial diagnoses as a part of an appointment 
so that the patient's file is up-to-date.
= As a nurse, I want to record patient checkup status information 
so that the rest of the medical team will be able to have vital statistics available.
= As a nurse, I want to be able to input lab test results 
so that the doctor has all the information needed to make an accurate diagnosis.
= As a nurse, I want to be able to input final diagnoses 
so that the patient’s record is up-to-date and allows for further follow-up if required.
= As a nurse, I want to be able to choose from a set of currently available tests 
so that I can order a test for a patient.
= As a nurse, I want to be able to order lab tests 
so that I will have relevant information needed to diagnose the patient.
